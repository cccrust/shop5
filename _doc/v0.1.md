# Shop5 v0.1 — CLI 基礎

Rust + SQLite 打造的網路商店 CLI 工具，支援多賣家商品管理、購物車與訂單系統。

## 技術棧

- **語言:** Rust (edition 2021)
- **資料庫:** SQLite (via rusqlite, bundled)
- **CLI 框架:** clap (derive 模式)
- **日期處理:** chrono
- **序列化:** serde / serde_json
- **錯誤處理:** anyhow
- **輸出著色:** colored

## 資料庫結構 (5 張表)

| 表名 | 說明 |
|------|------|
| `users` | 使用者帳號 (username, display_name, role, bio, avatar) |
| `products` | 商品 (seller_id, title, price, stock, status, sales_count) |
| `cart_items` | 購物車明細 (user_id, product_id, quantity, UNIQUE) |
| `orders` | 訂單主檔 (buyer_id, seller_id, status, total, note) |
| `order_items` | 訂單明細 (order_id, product_id, 下單時快取名稱/價格/小計) |

### 約束說明

- `users.role` 限定 `buyer` / `seller` / `admin`
- `products.status` 限定 `active` / `inactive` / `deleted`
- `orders.status` 限定 `pending` / `paid` / `shipped` / `delivered` / `cancelled`
- `cart_items` 使用 `UNIQUE(user_id, product_id)` + `ON CONFLICT DO UPDATE SET quantity = quantity + ?`
- 下單時快取商品名稱與價格到 `order_items`，避免歷史訂單受後續編輯影響
- 下單時自動扣庫存並增加 `sales_count`
- 訂單僅支援單一賣家（從購物車建立時自動檢查）

## CLI 指令

### 初始化

```
shop5 init
```

建立 `shop5.db`（可透過環境變數 `SHOP5_DB` 自訂路徑）。

### 使用者管理

```
shop5 user add <username> <display-name> [--role] [--bio]
shop5 user list [--search]
shop5 user get <id>
shop5 user update <id> [--name] [--bio] [--role]
shop5 user delete <id>
```

- `--role` 預設 `buyer`，可設為 `seller` / `admin`
- 角色不合法時回傳錯誤

### 商品管理

```
shop5 product add <seller-id> <title> <price> <stock> [--desc]
shop5 product list [--seller-id] [--status]
shop5 product get <id>
shop5 product update <id> [--title] [--price] [--stock] [--status] [--desc]
shop5 product delete <id>
```

- 價格與庫存不可為負數
- `price` 以整數儲存（單位：元/分），避免浮點數問題
- 商品 status 預設 `active`

### 購物車管理

```
shop5 cart add <user-id> <product-id> [--quantity]
shop5 cart remove <user-id> <product-id>
shop5 cart list <user-id>
shop5 cart clear <user-id>
```

- `--quantity` 預設 1，重複加入時累加數量
- 加入時檢查商品狀態 (`active`) 與庫存 (> 0)
- 列表顯示商品名稱、單價、數量、小計、總計

### 訂單管理

```
shop5 order create <buyer-id> [--note]     # 從購物車建立訂單
shop5 order buy <buyer-id> <seller-id> <product-id> <quantity> [--note]   # 直購
shop5 order list [--buyer-id] [--seller-id]
shop5 order get <id>
shop5 order update <id> --status <status>
```

- `create` 將購物車內容轉為訂單，自動清空購物車
- 下單前檢查庫存，不足時阻止下單
- `buy` 跳過購物車直接下單
- 訂單只接受單一賣家商品（混合賣家時回傳錯誤）

## 使用範例

```bash
# 初始化
shop5 init

# 建立使用者（賣家 + 買家）
shop5 user add alice 愛麗絲 --role seller --bio "3C 賣家"
shop5 user add bob 鮑勃 --role buyer --bio "愛買東西"

# 賣家上架商品
shop5 product add 1 "無線藍芽耳機" 599 50 --desc "高音質降噪"
shop5 product add 1 "USB-C 充電線" 199 100

# 買家加入購物車
shop5 cart add 2 1 --quantity 2
shop5 cart add 2 2 --quantity 3
shop5 cart list 2

# 下單
shop5 order create 2 --note "請包裝好"

# 賣家更新訂單狀態
shop5 order update 1 --status paid
shop5 order update 1 --status shipped
shop5 order update 1 --status delivered

# 檢視訂單
shop5 order get 1
```

## 建置

```bash
cargo build --release
```

執行檔位於 `target/release/shop5`。
