# v0.3 React 前端

## 目標
建構 Shop5 的 React 前端介面，支援使用者瀏覽商品、管理購物車與訂單。

## 技術棧
- **框架**: React 19 + TypeScript
- **建置工具**: Vite 6
- **樣式**: Tailwind CSS v4 (mobile-first, 深色主題)
- **路由**: React Router v7

## Production 模式

### 後端提供靜態檔案
當 `web --dev` 未指定時（production 模式），axum 會：

1. 將 `/assets` 路徑對應到 `web/dist/assets` 靜態目錄
2. 所有其他非 `/api` 路由回傳 `index.html`（SPA fallback）

### 建置與執行

```bash
cd web && npm run build   # 建置前端
./run.sh prod              # 啟動全端伺服器
```

`run.sh` 支援四種模式：

| 模式 | 說明 |
|------|------|
| `dev` | API 伺服器（`--dev`）+ 需另開 Vite dev server |
| `prod` | 全端 production 伺服器 |
| `api` | 僅 API 伺服器（等同 dev） |
| `build` | 建置前端 |

## 開發模式
在開發時，後端加上 `--dev` 參數則不提供靜態檔案，僅提供 API。
前端由 Vite dev server 負責（port 5173），開發體驗較佳：

```bash
./run.sh dev                # 啟動 API（背景）
cd web && npm run dev       # 啟動 Vite dev server
```

## 頁面功能

| 頁面 | 路由 | 說明 |
|------|------|------|
| 使用者列表 | `/users` | 列出所有使用者，點擊進入詳情 |
| 使用者詳情 | `/users/:id` | 查看與編輯使用者資料 |
| 商品列表 | `/` | 首頁，瀏覽與搜尋商品 |
| 商品詳情 | `/products/:id` | 查看商品、加入購物車 |
| 購物車 | `/cart` | 檢視購物車、調整數量、結帳 |
| 訂單列表 | `/orders` | 檢視歷史訂單 |
| 訂單詳情 | `/orders/:id` | 查看訂單明細、更新狀態 |

## 型別定義 （`types.ts`）

```typescript
interface User { id, username, display_name, role, bio, avatar, created_at, updated_at }
interface Product { id, seller_id, title, description, price, stock, status, sales_count, created_at, updated_at }
interface CartItem { id, user_id, product_id, quantity }
interface CartItemWithProduct extends CartItem { title, price, stock }
interface Order { id, buyer_id, seller_id, status, total, note, created_at, updated_at }
interface OrderItem { id, order_id, product_id, product_title, product_price, quantity, subtotal }
interface OrderWithItems { order: Order; items: OrderItem[] }
```
