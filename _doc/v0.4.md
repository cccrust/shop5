# v0.4 分類與搜尋

## 目標
新增商品分類系統與全文搜尋功能，讓使用者可以依分類瀏覽商品並進行關鍵字搜尋。

## 資料表異動

### 新增 categories
| 欄位 | 型態 | 說明 |
|------|------|------|
| id | INTEGER PK | |
| name | TEXT UNIQUE | 分類名稱 |
| parent_id | INTEGER FK→categories | 上層分類（支援樹狀結構） |

### products 新增欄位
- `category_id INTEGER REFERENCES categories(id) ON DELETE SET NULL`

## CLI 新增

```
shop5 category add <name> [--parent-id]
shop5 category list
shop5 category get <id>
shop5 category delete <id>

shop5 product add ... [--category-id]
shop5 product list ... [--category-id]
shop5 product search [--keyword] [--category-id] [--min-price] [--max-price] [--seller-id]
shop5 product update ... [--category-id]
```

## Web API 新增

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | `/api/categories` | 分類列表 |
| POST | `/api/categories` | 新增分類 |
| DELETE | `/api/categories/{id}` | 刪除分類 |
| GET | `/api/products/search?q=&category_id=&min_price=&max_price=&seller_id=` | 商品搜尋 |

## 前端新增

- **Search 頁面** (`/search`): 搜尋欄 + 分類下拉篩選 + 價格區間
- 底部導航列新增「搜尋」圖示
- 搜尋結果顯示商品名稱、價格、分類標籤

## 搜尋實作方式

- 使用 SQL `LIKE '%keyword%'` 比對商品標題與描述
- 支援分類篩選 (`category_id`)
- 支援價格範圍 (`min_price` / `max_price`)
- 支援賣家篩選 (`seller_id`)
- 僅搜尋 `status = 'active'` 商品
