# v0.6 評價系統

## 目標
訂單送達後，買家可以對商品進行星等評分（1–5）並撰寫評價內容；商品頁面顯示平均評分與評價列表。

## 資料表

```sql
CREATE TABLE reviews (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    order_id    INTEGER NOT NULL REFERENCES orders(id) ON DELETE CASCADE,
    user_id     INTEGER NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    product_id  INTEGER NOT NULL REFERENCES products(id) ON DELETE CASCADE,
    rating      INTEGER NOT NULL CHECK(rating >= 1 AND rating <= 5),
    content     TEXT DEFAULT '',
    created_at  TEXT NOT NULL DEFAULT (datetime('now')),
    UNIQUE(order_id, product_id, user_id)
);
```

`products` 表新增 `rating REAL DEFAULT 0` 與 `review_count INTEGER DEFAULT 0` 欄位，每次新增或刪除評價時自動更新。

## CLI 指令

```
shop5 review add <order_id> <user_id> <product_id> <rating> [--content]
shop5 review list <product_id>
shop5 review get <id>
shop5 review delete <id>
```

## Web API

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | `/api/reviews` | 新增評價 |
| GET | `/api/reviews/{id}` | 取得評價 |
| DELETE | `/api/reviews/{id}` | 刪除評價 |
| GET | `/api/reviews/product/{product_id}` | 商品評價列表 |
| GET | `/api/reviews/user/{user_id}` | 使用者評價列表 |

## 前端

- **ProductDetail**: 顯示評分星等 + 評價列表
- **OrderDetail**: 已送達訂單顯示「評價」按鈕，點擊可彈出評分選項與評價輸入
- **UserDetail**: 顯示該使用者寫過的評價列表

## 驗證邏輯

- 只能對已送達 (`delivered`) 的訂單評價
- 必須是訂單買家才能評價
- 商品必須在該訂單明細中
- 評分範圍 1–5（整數）
