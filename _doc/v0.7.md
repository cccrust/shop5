# v0.7 完整購物車流程

## 目標
購物車可調整數量、刪除單項、進入結帳頁面填寫備註後下單，並顯示訂單確認頁。

## Web API 新增

| 方法 | 路徑 | 說明 |
|------|------|------|
| PUT | `/api/cart/{user_id}/{product_id}` | 更新購物車數量 |
| POST | `/api/orders/preview` | 預覽訂單（不扣庫存） |

### PUT /api/cart/{user_id}/{product_id}
```json
{ "quantity": 3 }
```

### POST /api/orders/preview
```json
{ "buyer_id": 1 }
```
回傳商品明細、賣家資訊、總金額。

## 前端新增/修改

| 路由 | 頁面 | 說明 |
|------|------|------|
| `/cart` | Cart（強化） | 數量 +/- 按鈕、移除按鈕、連結結帳 |
| `/checkout` | Checkout（新增） | 訂單摘要 + 備註輸入 + 確認下單 |
| `/order/confirm/:id` | OrderConfirm（新增） | 訂單成功確認頁（綠色打勾 + 繼續購物） |

## 後端新增

- `model/cart::update_qty()` — 直接設定數量（非累加），檢查庫存
- `model/order::preview_from_cart()` — 列出購物車內容、驗證庫存、計算總額（不扣庫存）
