# v0.8 賣家儀表板

## 目標
賣家專屬儀表板，顯示銷售統計摘要、近 30 日營收趨勢圖、熱銷商品排行。

## Web API 新增

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | `/api/seller/{id}/stats` | 賣家銷售統計 |

### 回傳格式

```json
{
  "daily": [{ "date": "2026-06-14", "order_count": 3, "revenue": 2250 }],
  "top_products": [
    { "id": 2, "title": "耳機", "price": 100, "sales_count": 6, "total_revenue": 600 }
  ],
  "total_orders": 3,
  "total_revenue": 2250,
  "avg_order_value": 750
}
```

## 後端新增

- `model/stats.rs` — `SellerStats` 結構體 + `seller_stats()` 查詢函數
  - 每日銷售統計：`DATE(created_at)` GROUP BY + COUNT + SUM
  - 熱銷商品：按 `sales_count` 排序 LIMIT 10
  - 摘要：總訂單數、總營收、平均客單價

## 前端新增

| 路由 | 頁面 | 說明 |
|------|------|------|
| `/seller/dashboard` | SellerDashboard | 儀表板（3 張摘要卡 + 營收柱狀圖 + 熱銷排行） |

### 摘要卡
- 訂單數、總營收、平均客單價

### 營收趨勢圖
- 純 CSS 柱狀圖（無第三方圖表庫）
- 顯示近 14 天每日營收

### 熱銷商品排行
- 排名、商品名稱、售價、銷售量、營收

## 前端異動

- `UserDetail.tsx`: 賣家按鈕列新增「儀表板」按鈕
- `App.tsx`: 新增 `/seller/dashboard` 路由
